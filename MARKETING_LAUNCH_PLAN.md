- **Signups**: Target 1,000 signups in first 90 days
- **Conversion Rate**: Target 2-5% visitor-to-signup

### Activation Metrics
- **Trial-to-Activation**: Target 40%+ create first schedule
- **Time to First Value**: Target <10 minutes
- **Feature Adoption**: Track AI feature usage, analytics usage

### Revenue Metrics
- **Free-to-Paid Conversion**: Target 5%+ in first 90 days
- **Monthly Recurring Revenue (MRR)**: Target $5,000+ by month 3

- **Churn Rate**: Target <5% monthly

### Engagement Metrics
- **Daily Active Users (DAU)**: Target 30%+ of paid users
- **Weekly Active Users (WAU)**: Target 60%+ of paid users
- **Feature Usage**: Track key feature adoption
